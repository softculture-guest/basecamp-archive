RFA. Семейства в Revit

Продвинутый пользователь не может обойтись базовыми семействами программы: для работы над сложными проектами всегда приходится создавать или настраивать собственные. На курсе рассказываем полностью как создавать и использовать семейства в проекте.


Для чего нужен курс:

  • Для знакомства с темой, не заявляем, что он отображает решение каких-то конкретных рабочих запросов
  • Для создания сложных бим-семейств
  • Может классно продолжить Ревит для Интерьеров: можно создавать семейства для интерьерных проектов
Нужен чтобы создавать сложные семейства. Чтобы быстро смоделить не обращаясь за помощью к BIM менеджерам. Чтобы понимать как работают BIM менеджеры. Для дизайнеров интерьеров, чтобы создавать семейства свои. Мало можно найти в инете клевых в свободном доступе семейств.


Кому нужен курс:

  • Не для новичков, люди после наших курсов Ревит или с опытом работы в нем полгода-год, реализованные проекты
  • не новички в Revit
